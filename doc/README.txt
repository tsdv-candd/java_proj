================================================================================                        How to run the project
===============================================================================

Step1. Open project file by using eclipse 
    "Version: Kepler Service Release 2"
    Go to the folder ..\NRL\, you can see the project file ".project"
    Left click to open the project.

Step2. Running the project by
    Select "Run -> Run (Ctrl+F11)"

Step3. Interact with the program by refer to the requirement document
    1. Display Match Schedule
    2. Enter Round Results
    3. Display Ladder
    4. Team Results
    5. Exit Program

Note: 
    This project use input text files as input data.
    These text file is place at the folder named "Resource"
    ..\NRL\Resources
    You can change content of these text file as input test data.
